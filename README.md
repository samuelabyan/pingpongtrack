# PingPong Score Tracker
A mobile-first webapp for tracking table tennis matches among a group of players: [PingPong Score Tracker](https://samuelabyan.github.io/pingpongtrack/).
Supports quick games, knockout tournaments (1/8, 1/4, and semi-final formats), and round-robin leagues where every player faces every other player twice. Tracks wins, losses, points, and head-to-head records with a full statistics view, player comparison, and recent form streaks. All data is stored in the browser with JSON export/import. Features a clean light/dark theme and works entirely offline.
